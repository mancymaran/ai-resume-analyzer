# AI Resume Analyzer

A Python-based AI Resume Analyzer that allows users to upload resumes and get instant analysis using AI. Ideal for job seekers who want feedback on their resumes and tips to improve them.  

---

## Features

- Upload resumes in **PDF** or **DOCX** format  
- AI-powered analysis of **skills, experience, and formatting**  
- Easy-to-use web interface built with **Flask**  
- Instant feedback with clear suggestions  

---

## Installation

1. **Clone the repository**  

```bash
git clone https://github.com/mancymaran/ai-resume-analyzer.git
cd ai-resume-analyzer

Install dependencies
pip install -r requirements.txt

Run the app
python app.py

Open the browser
Go to: http://127.0.0.1:5000

Usage
Open the web page.
Upload your resume (PDF/DOCX).
Wait for the AI to analyze your resume.
View suggestions for improvements.

Upload Multiple Resumes
You can upload more than one resume to analyze them all at once:

Click the Upload Resume button.
Select one or more PDF/DOCX files.
Wait for the AI to process each resume.
View individual analysis results for each resume.
This allows you to compare multiple resumes or test updated versions easily.
