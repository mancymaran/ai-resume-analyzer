# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Flask, and spaCy.

## Features
- Upload PDF resume
- Extract resume text
- NLP-based processing
- Simple web interface

## Technologies Used
- Python
- Flask
- spaCy
- PyPDF2

## How to Run

pip install -r requirements.txt
python app.py

Open browser:
http://127.0.0.1:5000
